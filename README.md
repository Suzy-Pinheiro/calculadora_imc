
# 🧮 Calculadora de IMC

Aplicação de console desenvolvida em C# para cálculo do Índice de Massa Corporal (IMC).

## 📌 Sobre o projeto

Este programa permite que o usuário informe seu nome, peso e altura, realizando automaticamente o cálculo do IMC e exibindo a classificação de acordo com os padrões de saúde.

Além disso, o sistema também calcula a quantidade recomendada de água a ser consumida diariamente com base no peso informado.

## ⚙️ Funcionalidades

- Entrada de dados do usuário
- Conversão automática de altura (cm para metros)
- Cálculo do IMC
- Classificação por faixa:
  - Abaixo do peso
  - Peso normal
  - Sobrepeso
  - Obesidade Grau I
  - Obesidade Grau II
  - Obesidade Mórbida

 ## 💧 EXTRA
- Cálculo da ingestão diária de água
